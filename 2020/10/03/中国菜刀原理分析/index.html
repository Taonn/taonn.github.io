总结

  • 文件管理
    • key : 连接一句话木马 value: 解码为base64_decode
    • z0 -> 木马核心代码
    • z1 -> 目录信息
  • 虚拟终端
    • key : 连接一句话木马 value: 解码为base64_decode
    • z0 -> 木马核心代码
    • z1 -> cmd (windows) 或者 /bin/sh (Linux)
    • z2 -> 执行的命令
